## Quick Commands

- New note: Cmd+N
- Search: Cmd+O
- Write a title, then apply template: Cmd+R

## New note

- **`Cmd+N`** — create a new note. It lands in **`6. Zettelkasten
- Rename with **`F2`** (or click the title at the top of the note).

## Apply a template

1. Open or create the note you want to fill.
2. Press **`Cmd+R`** (the *Insert template* command) — or **`Cmd+P`** → "Insert template".
3. Pick a template from the list:
   - **Full Note** — general note: date/time, tags, title, references.
   - **Source Note** — for material you're reading: outline, key points, reflections.

## Other handy commands

- **`Cmd+O`** — quick switcher (jump to any note by name)
- **`Cmd+P`** — command palette (search every command)
- **`Cmd+Shift+F`** — search across the whole vault
- **`[[`** — start a link to another note
